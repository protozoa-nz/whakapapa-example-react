
# react-again

it's a real sweet app

[deployed here](http://protozoa.nz/whakapapa-example-react/)

## how

### install

```
git clone https://github.com/pietgeursen/react-again
cd 
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
